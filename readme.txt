Deskripsi singkat : Program yang dapat menyimpan nilai siswa dengan sistem keamanan sehingga selain admin tidak ada yang dapat mengetahui password program, kecuali diberitahu oleh Admin. Admin dapat mengubah dan melihat nilai, sedangkan pengguna biasa hanya bisa melihat nilai saja. Dalam aplikasi juga terdapat pilihan untuk menutup program.


**********************************************  Cara kerja singkat   ****************************************************

Menambah nilai (sebagai admin) :
1. Pilih pilihan 1 pada menu utama
2. Masukkan password (default = 2707)
3. Masukkan data yang ada, misalnya nama, lalu tekan enter
4. Akan ada terdapat preview data yang akan diinput. Jika sudah yakin, pilih 1, jika ingin ulang, pilih 2.
5. Jika sudah, akan kembali ke halaman awal

Menghapus nilai (sebagai admin) :
1. Pilih pilihan 3 pada menu utama
2. Masukkan password (default = 2707)
3. Masukkan nama yang ingin dihapus lalu diikuti oleh semesternya (misalnya Putra1)
4. Nilai pada semester tersebut akan dihapus lalu kembali ke menu utama

Melihat nilai :
1. Pilih pilihan 2 pada menu utama
2. Masukkan nama siswa yang ingin dicari, lalu tekan enter
3. Masukkan semester yang ingin dicari, lalu tekan enter
4. Akan muncul daftar nilai siswa tersebut, jika sudah selesai tekan enter
